# Turtle Character Drawing Project

## Description

This is a Python Turtle Graphics project that allows users to draw different cartoon characters, animals, and objects by entering their names.

The project uses Python functions, modules, user input, and Turtle Graphics to create drawings.

## Features

### Cartoon Characters

- Doraemon
- Pikachu
- Shinchan
- Nobita
- Hello Kitty
- Mickey Mouse
- Minion
- SpongeBob
- Totoro
- Smiley Face

### Animals

- Panda
- Cat
- Dog
- Bear
- Rabbit

### Objects

- House
- Star
- Heart
- Flower
- Tree
- Rocket
- Robot
- Car
- Boat
- Ice Cream



## Requirements

- Python 3.x
- Turtle Module (built into Python)

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:

python project.py

4. Enter the name of a character, animal, or object.
5. The Turtle Graphics window will open and display the drawing.
